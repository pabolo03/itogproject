## Описание программы
Приложение «GitHub User Finder» позволяет искать пользователей GitHub через официальный API, просматривать их основную информацию и сохранять избранных пользователей в локальный JSON‑файл.

## Функциональность

- Поиск пользователей GitHub по имени пользователя
- Отображение основной информации о пользователе:
  - Имя
  - Количество публичных репозиториев
  - Количество подписчиков
  - Местоположение
  - Биография
- Добавление пользователей в избранное
- Просмотр списка избранных пользователей
- Сохранение избранных пользователей в JSON‑файл
- Валидация ввода (поле поиска не может быть пустым)

## Использование API GitHub

Приложение использует официальный GitHub API (v3) для получения информации о пользователях. Базовый URL: `https://api.github.com/users/{username}`.

API возвращает JSON‑объект с информацией о пользователе. Приложение обрабатывает следующие поля:
- `login` — имя пользователя
- `name` — полное имя
- `public_repos` — количество публичных репозиториев
- `followers` — количество подписчиков
